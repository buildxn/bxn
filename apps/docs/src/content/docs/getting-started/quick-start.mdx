Get up and running with bxn http in just a few minutes.

## Create a New Project

The fastest way to get started is using the `create-bxn` init command:

<Tabs syncKey="pkg">
<TabItem label="pnpm">

```bash
pnpm create bxn@latest
```

</TabItem>
<TabItem label="npm">

```bash
npm create bxn@latest
```

</TabItem>
<TabItem label="yarn">

```bash
yarn create bxn@latest
```

</TabItem>
</Tabs>

Once created, start the development server:

<Tabs syncKey="pkg">
<TabItem label="pnpm">

```bash
cd your-project-name
pnpm dev
```

</TabItem>
<TabItem label="npm">

```bash
cd your-project-name
npm run dev
```

</TabItem>
<TabItem label="yarn">

```bash
cd your-project-name
yarn dev
```

</TabItem>
</Tabs>

Visit `http://localhost:3000` in your browser to see your API running!

## Manual Installation

If you prefer to set up manually, install both the framework and CLI:

<Tabs syncKey="pkg">
<TabItem label="pnpm">

```bash
pnpm add @buildxn/http
pnpm add -D bxn
```

</TabItem>
<TabItem label="npm">

```bash
npm install @buildxn/http
npm install -D bxn
```

</TabItem>
<TabItem label="yarn">

```bash
yarn add @buildxn/http
yarn add -D bxn
```

</TabItem>
</Tabs>

Then add these scripts to your `package.json`:

```json
{
  "scripts": {
    "dev": "bxn http start --watch",
    "start": "bxn http start",
    "build": "tsc"
  }
}
```

## Create Your First Route

Create a simple route file at `src/routes/get.ts`:

```typescript
import { json, type RequestHandler } from '@buildxn/http';

const handler: RequestHandler = () => {
  return json({
    message: 'Hello, World!',
    timestamp: new Date().toISOString()
  });
};

export default handler;
```

## Run the Development Server

Start the development server with watch mode for auto-reload:

```bash
npx bxn http start --watch
```

Your API is now running at `http://localhost:3000`!

## Test Your API

Open your browser or use curl to test the endpoint:

```bash
curl http://localhost:3000
```

You should see:

```json
{
  "message": "Hello, World!",
  "timestamp": "2024-01-15T10:30:00.000Z"
}
```
